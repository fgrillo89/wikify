# Materials Science Field Guide

Structure: Introduction (application gap) -> Methods -> Results+Discussion -> Conclusion.
Citations: [1] numbered (ACS) or author-year (Elsevier). Primary sources.
Figures: XRD, SEM/TEM, XPS, IV curves. Scale bars, axis labels, error bars required.

Rules: open with the application need, not history. Report all process parameters (temperature, pressure, precursor, pulse/purge times, substrate). Characterization order: structure then composition then properties. SI units: nm for thickness; C or K for temperature; Pa or Torr for pressure (pick one each). Compare quantitatively against prior work with citation and exact numbers. Captions are self-contained (material, technique, key observation). Link processing to microstructure to properties. Report sample counts, standard deviations, batch variation.
