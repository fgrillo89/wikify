# Grant Proposal — Writing Guide

## Exemplary Models

NIH R01 proposals, NSF standard proposals, and ERC Starting/Consolidator Grants are
canonical models. The NIH "Specific Aims" format is widely adopted even outside
biomedical fields due to its clarity and persuasive structure.

## Structure

1. **Abstract** — Project summary for reviewers and program officers. Must cover the
   problem, approach, and expected impact in one page or less.
2. **Specific Aims** — One page. Hook (1-2 sentences on the problem), gap, long-term
   goal, objective of this project, central hypothesis, 2-3 specific aims with brief
   rationale for each.
3. **Significance** — Why does this matter? What gap does it fill? What will change
   if the project succeeds?
4. **Innovation** — What is new about the approach, methods, or concepts?
5. **Approach** — Detailed methods per aim. Include preliminary data, expected
   outcomes, potential pitfalls and alternative strategies.

## Key Conventions

- **Tone**: Confident but not arrogant. Use "We will" not "We hope to." Avoid hedging
  language except when describing alternative approaches.
- **Specific Aims page** is the most important page in the proposal. Many reviewers
  form their opinion from this page alone. It must be compelling and self-contained.
- **Preliminary data** demonstrates feasibility. Even a small pilot result dramatically
  strengthens the proposal.
- **Pitfalls and alternatives** show maturity. For each aim, state what could go wrong
  and what backup plan exists.

## Actionable Instructions for the LLM

- The Specific Aims page must follow the hook-gap-goal-hypothesis-aims structure.
- Each aim should be testable and have a clear deliverable.
- Significance should cite statistics or evidence showing the problem's importance.
- Innovation should explicitly state what has NOT been done before.
- Approach subsections should follow: rationale, methods, expected outcomes,
  pitfalls/alternatives.
- Never use tentative language ("We might," "This could potentially") for the core
  plan. Reserve hedging for alternative strategies only.
