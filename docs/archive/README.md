# Archive

This directory is the single archive root for documentation that is no longer
part of the current working docs surface.

Current docs to use first:

- `docs/architecture.md`
- `docs/project-status.md`
- `docs/refactor/wiki-deep-refactor-plan.md`
- `docs/design/wiki-runtime-refactor-plan.md`

Archive layout:

- `docs/archive/refactor-plan.md`
  Original completed repo refactor plan kept for historical context.
- `docs/archive/design/`
  Archived design docs that were once current but are no longer part of the
  active docs surface.
- `docs/archive/design/legacy/`
  Older design archive material that previously lived under
  `docs/design/archive/`.

Archived docs are kept for historical context, prior design thinking, and
implementation archaeology. They should not be treated as current architecture
or current execution plans unless they are explicitly restored.
