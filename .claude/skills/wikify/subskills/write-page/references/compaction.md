# Compaction

Optional stage for consolidating extracted notes before writing.

Use when a concept has many overlapping evidence entries or extracted
notes.

Produce a compact dossier with:

- One definition.
- 3-5 sentence summary.
- Deduplicated parameters.
- Deduplicated mechanisms.
- Deduplicated relationships.
- Deduplicated equations.
- Top evidence from diverse documents.

Do not discard contradictory evidence. Mark it for the workflow or
editor brief.
