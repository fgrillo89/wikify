# Editor Brief

Optional stage for turning a dossier into a section plan before writing.

The brief should decide:

- Register and tone.
- Lead paragraph instruction.
- Section headings.
- Evidence markers each section must cite.
- Contested versus established versus frontier claims.
- Figures/equations to include.
- Target length.
- Neighbor pages to avoid duplicating.

The brief is a planning artifact. It is not the final article.
