# CLI Grammar

Seven nouns are the stable agent-facing grammar:

```text
wikify corpus
wikify run
wikify work
wikify draft
wikify wiki
wikify render
wikify eval
```

Common verbs:

```text
init, show, list, find, add, set, build, check, commit, tend, close
```

Use actual `--help` output as the source of truth for flags. Do not add
aspirational examples that the CLI does not support.
